# AI 뉴스 스타일 변환 플랫폼

## 프로젝트 개요

뉴스는 중요한 정보를 담고 있지만, 형식이 딱딱하고 어렵게 느껴져 많은 사람들이
읽기를 꺼려한다. 특히 청소년이나 뉴스에 익숙하지 않은 사람들에게는 더욱 그렇다.
이 프로젝트는 그러한 문제를 해결하기 위해 기획되었다.

이 서비스는 뉴스 기사의 핵심 내용을 유지하면서, 사용자가 선택한 스타일로 변환해
제공하는 AI 기반 웹 플랫폼이다. 동화체, 소설체, 카드 요약 등 다양한 형식을
지원하며, 누구나 자신에게 맞는 방식으로 뉴스를 접할 수 있도록 한다.

---

## 주요 기능

### 자동 추천 모드

AI가 오늘의 주요 뉴스를 자동으로 선별하고 변환하여 피드 형태로 제공한다.
별도의 입력 없이 원하는 스타일 탭을 선택하는 것만으로 뉴스를 읽을 수 있다.
미리 검증이 완료된 콘텐츠가 제공되기 때문에 정확도가 높다.

### 직접 변환 모드

사용자가 직접 관심 있는 뉴스 기사의 URL을 입력하면 AI가 실시간으로 원하는
스타일로 변환해 준다. 자동 추천에 없는 기사도 변환할 수 있어 활용 범위가 넓다.

---

## 기술적 특징

이 서비스의 핵심은 두 AI 에이전트가 협력하는 구조에 있다. 변환을 담당하는 AI가
기사를 선택된 스타일로 재구성하면, 검증을 담당하는 별도의 AI가 원문과 비교하여
내용이 누락되거나 왜곡되지 않았는지 확인한다. 문제가 발견될 경우 자동으로 재생성을
요청하거나 수정 방향을 제시하는 방식으로 정확성을 보장한다.

형식은 바뀌어도 내용은 지킨다는 원칙 아래, 더 많은 사람이 뉴스를 쉽고 재미있게
접할 수 있는 환경을 만드는 것이 이 프로젝트의 목표이다.
